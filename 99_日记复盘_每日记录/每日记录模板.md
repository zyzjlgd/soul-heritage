## 📊 七、Dataview 汇总查询（演示）

> 下面的查询语句可查看历史修行数据。复制到 Obsidian 任意笔记即可使用。

### ① 查本月练功总览
```dataview
TABLE 练功, 身体状态, 感悟主题
FROM ""
WHERE date >= 2026-04-01 AND date <= 2026-04-30
SORT date DESC
```

### ② 查某功法历史记录
```dataview
TABLE date, 练功
FROM ""
WHERE contains(练功, "升阳功")
SORT date DESC
```

### ③ 查守觉做得最好的几天
```dataview
TABLE date, 守觉场景, 觉醒程度
FROM ""
WHERE contains(守觉场景, "处事")
SORT date DESC
LIMIT 10
```

### ④ 查本月觉醒感悟
```dataview
TABLE date, 感悟主题
FROM ""
WHERE contains(感悟主题, "觉醒")
SORT date DESC
```
