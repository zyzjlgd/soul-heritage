# 自动化工具说明

> 这是一份记录当前使用的自动化工具配置文档。不用每次重复操作。

---

## 已配置的自动化

### Dataview（已启用）

用于横向统计，例如：

- 查"哪天练了B功法"：`dv.pages('#升阳功').where(t => t.file.day == '某日')`
- 查"全部#天赋标签"：`dv.pages('#天赋')`
- 查"某月全部感悟"：`dv.pages('#日记').where(t => t.file.ctime.month == 12)`

### Obsidian模板（已配置）

- 模板文件夹：`99_日记复盘_每日记录/templates`
- 快捷键：左侧边栏新建按钮 → 选「每日记录模板」
- 日期格式：`YYYY-MM-DD`
- 自动填入：`{{date}}`变量 + 当前日期

### 自动插入模板操作步骤

1. Obsidian左侧边栏点击「新建」按钮
2. 选择「每日记录模板」
3. 自动生成带日期和元数据的空白日记
4. 填写内容即可
